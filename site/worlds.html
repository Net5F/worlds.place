Amalgam Engine Worlds

Explore this page to find worlds built by the Amalgam Engine community.

Note: On Windows, you may see "Microsoft Defender found threats" when you try to run a client. This happens until a certain number of people run the executable. To get around it, go to "Virus & threat protection" -> "Protection history", click "Threat blocked" and select "Actions" -> "Allow". To ensure this is safe, only download from a project's official Github page or website.
