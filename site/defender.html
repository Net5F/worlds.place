Microsoft Defender issues

On Windows, you may see "Microsoft Defender found threats" when you try to run a client. This happens until a certain number of people run the executable.

To get around it, go to "Virus & threat protection" -> "Protection history", click "Threat blocked" and select "Actions" -> "Allow".

To ensure this is safe, only download from a project's official Github page or website.

Developers

Developers can try to avoid Microsoft Defender false positives by triggering the warning once, then using this page to register the file with Microsoft.
